# 发布前检查清单

## 自动验证

- [ ] 运行 `npm test`，确认账本工具、云函数核心逻辑、图表模型、视觉脚本核心逻辑全部通过。
- [ ] 运行 `npm run lint`，确认项目结构、脚本语法、tabBar 图标、主包 2MB 限制、任务工作流和敏感配置扫描通过。
- [ ] 运行 `VERIFY_MODE=full ./init.sh`，确认标准入口可从干净会话重新执行。

## 小程序手工验收

- [ ] 微信开发者工具导入 `/Users/Lin/project/shasha` 并重新编译。
- [ ] 首页能展示云同步状态、月度摘要和最近流水。
- [ ] 记账页可新增今日/昨日/自定义日期账单，金额以“分”为单位落库。
- [ ] 明细页可按收支、分类、账户、标签、日期区间筛选。
- [ ] 明细页可编辑、删除记录，刷新后数据仍与云端一致。
- [ ] 图表页可切换日/周/月趋势，可切换支出/收入分类占比。
- [ ] 空数据、单条数据、大量数据状态无白屏、溢出或遮挡。

## 云开发与权限

- [ ] 云函数 `initUser`、`ledgerQuery`、`ledgerWrite` 均已上传并部署。
- [ ] 集合 `users`、`ledger_records`、`categories`、`accounts`、`budgets`、`tags` 权限保持“所有用户不可读写”，客户端不直连跨用户数据。
- [ ] `ledger_records` 新增记录包含当前用户 `openid`。
- [ ] `ledgerWrite` 使用非法金额、非法日期、缺少 ID、非本人记录时返回可读错误。
- [ ] 发布前未提交 AppSecret、支付密钥、云环境私密配置、CI 私钥。

## 预览与上传

- [ ] 微信开发者工具生成预览码，在模拟器和真机至少各跑一遍核心流程。
- [ ] 如后续接入 `miniprogram-ci`，预览或上传脚本必须继续复用 `./init.sh` 的验证入口。
