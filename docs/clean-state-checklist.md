# 收尾检查清单

结束一轮会话前过一遍：

- [ ] `docs/feature_list.json` 真实反映当前功能状态。
- [ ] 同一时间最多只有一个功能是 `in_progress`。
- [ ] `passing` 功能都有验证证据。
- [ ] `docs/coding-progress.md` 已记录本轮目标、改动、验证、风险和下一步。
- [ ] 云函数、数据库集合、权限规则的变更已说明。
- [ ] 没有提交微信 AppSecret、私钥、支付密钥、云环境敏感配置。
- [ ] `./init.sh` 已运行，或已记录未运行原因。
- [ ] 下一轮会话可以只靠仓库内文件继续推进。
